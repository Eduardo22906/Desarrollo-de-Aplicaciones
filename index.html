Docente: Ramón Puente Eduardo Fredy

Bienvenido al Inicio

Introducción

La asignatura de Programación de Aplicaciones Móviles Web tiene como objetivo principal proporcionar a los estudiantes los conocimientos y habilidades necesarios para diseñar y desarrollar aplicaciones móviles web utilizando tecnologías y frameworks actuales.

Objetivos

  1. Conocer los conceptos fundamentales de la programación de aplicaciones móviles web.
  2. Aprender a diseñar y desarrollar aplicaciones móviles web utilizando tecnologías y frameworks actuales.
  3. Entender la importancia de la experiencia del usuario en la creación de aplicaciones móviles web.
  4. Desarrollar habilidades para resolver problemas y depurar código en entornos de desarrollo integrados (IDEs).

Contenidos

  1. Introducción a la programación de aplicaciones móviles web.
  2. Lenguajes de programación para aplicaciones móviles web (HTML, CSS, JavaScript).
  3. Frameworks para aplicaciones móviles web (React Native, Ionic, etc.).
  4. Diseño de interfaces de usuario para aplicaciones móviles web.
  5. Desarrollo de aplicaciones móviles web utilizando APIs y bases de datos.
  6. Pruebas y depuración de aplicaciones móviles web.
  7. Publicación y distribución de aplicaciones móviles web.

Evaluación

La evaluación de esta asignatura se realizará a través de:

  • Exámenes parciales y finales.
  • Proyectos prácticos individuales y grupales.
  • Participación y entrega de actividades semanales.
